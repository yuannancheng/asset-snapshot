## 现有功能总结

- 本地优先的 Tauri + React 资产快照应用，数据存储在 SQLite 单文件中。
- 支持平台和账户配置，账户类型固定为流动资产、非流动资产、负债。
- 支持新建、编辑、删除历史快照，汇总数据实时计算不入库。
- 支持停用账户；无历史数据的账户可以删除，无历史数据的平台可以连同子账户删除。
- 支持数据文件管理：查看当前 SQLite 路径、选择/新建数据文件、导出备份，并记住最近使用的数据文件。
- 日期选择器支持直接输入、日历选择、年月切换，非法日期失焦后标红提示。
- 仪表盘展示总资产、可用资产、启用账户数、趋势图、平台分布图和最近历史快照。
- 趋势图支持最近 3 个月、最近 1 年、最近 3 年和自定义区间，并按范围自动使用日、周、月粒度。
- 已开始使用 Git；当前环境的 `.git` 是只读挂载点，临时使用 `.git-store` 作为分离 Git 目录。
